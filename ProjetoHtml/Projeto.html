Introdução

Nesta aula aprendi sobre a estrutura básica do html e algumas tags:

Aprendi que todo o código html deve começar com a tag responsal por ajudar os navegadores identificar o tipo do documento feito em html "!DOCTYPE html".

A tag "< b > ... < /b > ou < strong> ... < /strong>" nos ajuda a destacar um texto ou seja negritar uma palavra Aprender a fazer titoy.

A tag "< head > < /head >" é uma tag que nos ajuda colocar um text por sima dos texto encontrado na tag < body > < /body >, comumente usado no seu interior a tag < title> para dar título a nossa página< /title>
